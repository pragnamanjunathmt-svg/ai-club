## Overview
NovaNews is a simple news application that shows latest news and also gives small AI summaries for each article. The main idea is to make news reading faster and easier.

## Features
- Shows latest global news
- Search news by category or keyword
- AI generated short summary (2 lines)
- User login system (basic)
- Save articles (My Library)

## Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Python (Flask)  
Database: PostgreSQL  
APIs: News API, Gemini API  

## How to Run the Project

1. Clone the repository  
git clone https://github.com/pragnamnajunathmt/ai-club

2. Go to project folder  
cd ai-club  

3. Install required packages  
pip install -r requirements.txt  

4. Create a .env file and add your API keys  
NEWS_API_KEY=your_api_key  
GEMINI_API_KEY=your_api_key  

5. Run the application  
python app.py  

6. Open in browser  
http://localhost:5000  

## Usage
- Open the homepage to see latest news  
- Use search to filter news  
- Click on summary button to get AI summary  
- Login to save articles

This project is made for AI & Software Guild recruitment. Some features might be basic but I tried to implement the core functionality and learn new concepts during the process.
